# 🧩 MARD 拼豆图纸生成器 — MARD Perler Bead Pattern Generator

> 上传一张图片，自动映射为 MARD 标准拼豆色号的图纸，每个格子标注色号，打印即可拼豆。
>
> Upload any image — automatically maps to MARD standard bead colors, labels every cell with its color code, ready to print and fuse.

---

## ✨ 功能 Features

| 功能                     | 说明                                                       |
| ------------------------ | ---------------------------------------------------------- |
| 📷 上传图片               | 点击或拖拽上传 JPG / PNG / WebP                            |
| 🔢 网格密度 4×4 ~ 100×100 | 滑块实时调整，保持宽高比                                   |
| 📏 格子大小 6~40px        | 控制每个色块的显示尺寸                                     |
| 🎨 色板切换               | 标准 221 色 ↔ 全部 291 色（含 P/Q/R/T/Y/ZG 系列）          |
| 🔄 旋转 / 镜像            | 左旋 90°、右旋 90°、水平翻转                               |
| 🎯 颜色限制               | 开启后只使用用量最多的 N 种颜色（减少买豆成本）            |
| ⬜ 素色模式               | 黑白线稿 + 色号标注，省墨环保打印                          |
| 🔲 网格线开关             | 显示/隐藏色块之间的分割线                                  |
| 🏷️ 色号标注               | 每个格子上显示 MARD 色号（如 A1、C15），深底白字、浅底黑字 |
| 📋 颜色图例               | 底部显示用到的所有色号 + 色块 + 用量                       |
| ⬇ 下载 PNG               | 高清图纸，底部附带尺寸、颜色数、颜色清单                   |
| 🖨️ 导出 PDF               | 浏览器打印预览，一键打印为一页                             |

## 🚀 使用方式 Usage

**单文件，零依赖。** 双击 `index.html` 即可在浏览器打开，无需安装任何东西。

1. 点击上传区域或拖拽一张图片
2. 调节网格密度、格子大小到满意
3. 可选：切换色板、限制颜色数、素色模式、隐藏网格线
4. 点击「标准图纸 PNG」下载图纸，或「导出 PDF」打印

## 🎨 MARD 色板 Palette

- **标准 221 色**：A、B、C、D、E、F、G、H、M 系列 — 日常拼豆常用
- **全部 291 色**：额外包含 P、Q、R、T、Y、ZG 系列 — 更多色彩选择

颜色映射使用加权 RGB 距离算法，自动为每个像素找到最接近的 MARD 色号。

## 📄 下载文件内容

下载的 PNG / PDF 包含：

- 拼豆图纸网格（每格标注色号）
- 成品尺寸信息（像素数、颜色数）
- 🎨 **颜色清单**：用到的色号 + 颜色方块 + 所需数量 — 方便采购

## 💡 小技巧 Tips

- **减少颜色数**：开启颜色限制，用色越少越省钱
- **素色模式**：黑白线稿更省墨，适合大量打印
- **网格线**：小格子时建议显示，剪裁更方便
- **建议密度**：32×32 ~ 60×60 适合大多数图片

## 🛠️ 技术说明 Tech

- 纯 HTML / CSS / JavaScript，单文件
- 核心算法：离屏 Canvas → 最近色映射 → 加权 RGB 距离 → 颜色量化
- 全部在浏览器端运行，**不上传任何图片到服务器**
